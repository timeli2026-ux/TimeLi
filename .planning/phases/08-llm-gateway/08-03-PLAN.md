  <name>Task 3: Create secure LLM logging service</name>
  <files>src/lib/logging/llm-logger.ts</files>
  <action>
Create secure logging service for LLM requests (SEC-16 compliance).

Export:
- logLLMRequest(params: LLMLogParams): void
- sanitizeForLogging(data: unknown): unknown

LLMLogParams:
{
  userId: string,
  endpoint: string,
  provider: string,
  inputTokens: number,
  outputTokens: number,
  cached: boolean,
  latencyMs: number,
  error?: string
}

Security requirements:
- NEVER log API keys (sanitize all headers)
- NEVER log full prompts (only endpoint + token counts)
- NEVER log PII from user messages (only metadata)
- Hash userId for logs (SHA-256 first 8 chars)
- Use structured JSON format for parseability
- Log level: info for success, warn for slow (>2s), error for failures

Sanitize function should:
- Remove any key containing 'key', 'secret', 'token', 'password', 'auth'
- Truncate long strings to first 50 chars + "..."
- Replace email patterns with "[EMAIL]"
  </action>
  <verify>npm run build passes</verify>
  <done>Secure LLM logger exports logLLMRequest and sanitizeForLogging with SEC-16 compliance</done>
