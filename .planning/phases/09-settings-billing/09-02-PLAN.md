<task type="auto">
  <name>Task 2: Implement Stripe webhook handler with signature verification</name>
  <files>src/app/api/webhooks/stripe/route.ts</files>
  <action>
Create POST handler for Stripe webhooks:
1. Get raw body with `request.text()` (CRITICAL for signature verification)
2. Get signature from `headers().get('stripe-signature')`
3. Verify with `stripe.webhooks.constructEvent(body, signature, STRIPE_WEBHOOK_SECRET)`
4. Handle events:
   - `checkout.session.completed`: Link Stripe customer to user, create subscription record
   - `customer.subscription.created`: Set status to 'trialing', record trial_start/trial_end
   - `customer.subscription.updated`: Update status (trialing→active, active→past_due, etc.), update period dates
   - `customer.subscription.deleted`: Set status to 'canceled'
   - `invoice.paid`: Log successful payment (for future analytics)
   - `invoice.payment_failed`: Set status to 'past_due'
5. Return 200 for handled events, 400 for verification failures

Log event type and subscription ID (not full payload - SEC-16). Use secure logging pattern from Phase 8.
  </action>
  <verify>Route exists, handles POST requests, rejects invalid signatures</verify>
  <done>Webhook handler verifies signatures and updates subscription status</done>
</task>
