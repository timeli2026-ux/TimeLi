Create POST endpoint that processes student onboarding data:

1. Authentication:
   - Get user from Supabase auth
   - Return 401 if not authenticated

2. Request validation with Zod:
   ```typescript
   const studentOnboardingSchema = z.object({
     preferences: z.object({
       timezone: z.string().min(1).max(100),
       sleepStart: timeSchema,
       sleepEnd: timeSchema,
     }),
     courses: z.array(z.object({
       name: z.string().min(1).max(100),
       semester: z.string().min(1).max(50),
       instructor: z.string().max(100).optional(),
       color: z.string().regex(/^#[0-9A-Fa-f]{6}$/),
       schedule: z.array(z.object({
         day: z.number().min(0).max(6),
         start: timeSchema,
         end: timeSchema,
       })).min(1),
     })),
     assignments: z.array(z.object({
       title: z.string().min(1).max(200),
       type: assignmentTypeEnum,
       dueDate: z.string().datetime(),
       estimatedHours: z.number().min(0.5).max(100),
       courseLocalId: z.string().optional(),  // maps to course in courses array
       notes: z.string().max(1000).optional(),
     })),
   })
   ```

3. Database operations (in transaction-like sequence):
   a. Upsert user_preferences with timezone and sleep times
   b. Insert courses, capture mapping of local IDs to DB IDs
   c. Insert assignments with mapped course_id (nullable if no course)
   d. Update profiles.onboarding_completed = true

4. Error handling:
   - Wrap in try/catch
   - Log errors
   - Return appropriate status codes

5. Response:
   ```typescript
   return NextResponse.json({
     success: true,
     coursesCreated: courses.length,
     assignmentsCreated: assignments.length,
   })
   ```

Use sanitizeUserInput for all string fields.
Follow pattern from src/app/api/onboarding/complete/route.ts.

   ```typescript
   if (isLastStep) {
     setIsSubmitting(true)
     try {
       // Map local course IDs to courseLocalId for assignments
       const assignmentsPayload = formData.assignments.map(a => ({
         title: a.title,
         type: a.type,
         dueDate: a.dueDate,
         estimatedHours: a.estimatedHours,
         courseLocalId: a.courseId,  // local ID, API will map to DB ID
         notes: a.notes,
       }))

       const response = await fetch('/api/onboarding/student/complete', {
         method: 'POST',
         headers: { 'Content-Type': 'application/json' },
         body: JSON.stringify({
           preferences: {
             timezone: formData.timezone,
             sleepStart: formData.sleepStart,
             sleepEnd: formData.sleepEnd,
           },
           courses: formData.courses.map(c => ({
             localId: c.id,  // for assignment mapping
             name: c.name,
             semester: c.semester,
             instructor: c.instructor,
             color: c.color,
             schedule: c.schedule,
           })),
           assignments: assignmentsPayload,
         }),
       })

       if (!response.ok) {
         const data = await response.json()
         throw new Error(data.error || 'Failed to complete setup')
       }

       // Redirect to dashboard on success
       window.location.href = '/dashboard'
     } catch (err) {
       setError(err instanceof Error ? err.message : 'Something went wrong')
       setIsSubmitting(false)
     }
   }
   ```
