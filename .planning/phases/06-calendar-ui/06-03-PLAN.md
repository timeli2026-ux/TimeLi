Implement native HTML5 drag and drop (no external library needed):

**CalendarEvent updates:**
- Add draggable={!event.isLocked} attribute
- onDragStart: Store event data in dataTransfer
- Visual feedback: opacity-50 while dragging
- Change cursor to grabbing while dragging

**WeekGrid updates:**
- Each 15-min slot is a drop target
- onDragOver: preventDefault() to allow drop, highlight valid drop zone
- onDrop: Calculate new time slot from drop position
- Snap to 15-minute grid (use alignToGrid from types.ts)
- Prevent dropping locked events
- Visual drop indicator: Show ghost of event at drop position

**API endpoint (src/app/api/schedule/update/route.ts):**
POST /api/schedule/update
Body: { eventId: string, newSlot: TimeSlot }
Response: { success: boolean, updatedEvent: ScheduleEvent }

Logic:
- Validate user owns the event (check against schedule)
- Validate new slot doesn't conflict with locked events
- If conflict with another goal event, warn (but allow - they can overlap for now)
- Update the event's slot in memory/state
- Return updated event

For now, store schedule in component state. Database persistence will come in a future phase.

**Optimistic update:**
- Update UI immediately on drop
- Show toast notification: "Event moved to [new time]"
- If API fails, revert position and show error toast

Install sonner for toast notifications:
```bash
npx shadcn@latest add sonner --yes
```
Add Toaster to the app layout.
